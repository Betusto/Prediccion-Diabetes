# Predicción Diabetes
Programa de escritorio que ayuda a predecir si una mujer tiene diabetes por medio de ciertos atributos, gracias a una red neuronal entrenada con datos de casos reales ocurridos en hospitales de la India y Alemania. 2019.<br>
Lenguaje usado: Java (Para predecir) y Python (Para entrenar la red neuronal).
